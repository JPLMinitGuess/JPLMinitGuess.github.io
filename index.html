MinitGuess

Automating Initial Guess Generation in Mystic

Welcome to the MinitGuess Homepage!

MinitGuess is an open source software toolbox designed to allow the automation of initial guess generation for orbital mechanics problems. Containing pre-computed data and models along with sample scenarios, the software allows for orbit creation and dynamical model integration in medium fidelity for feasibility analysis and export to higher fidelity models.

Features

  1. A Dynamical System Exploration Toolbox
  2. Dynamical Model Set: includes CR3BP, EH3BP, etc.
  3. Orbit Database: pre-computed orbits (periodic, quasi-periodic, etc.)
  4. Object-oriented approach for easy code reuse and maintenance
  5. Graphical User Interface (GUI) or Command-Line Interface (CLI) available
